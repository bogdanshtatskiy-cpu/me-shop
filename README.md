# 🛒 МЭ Магазин (OpenComputers + Firebase)

Полностью автоматизированный магазин для мода Applied Energistics 2, управляемый через компьютер из OpenComputers с облачной синхронизацией и мобильной веб-админкой.

## 🌟 Главные фичи

- **Клиент-серверная архитектура:** Данные магазина хранятся в Google Firebase Realtime Database.
- **Веб-панель управления:** Меняйте цены, управляйте балансом игроков и читайте логи прямо со своего смартфона через GitHub Pages.
- **Фоновая синхронизация:** Изменения в веб-панели появляются в игре автоматически (опрос БД каждые 15 секунд).
- **Корзина покупок:** Возможность набрать несколько товаров и оплатить всё одним кликом.
- **Умный сканер:** Добавление товаров в игру происходит простым помещением предмета в сундук — скрипт сам определяет NBT, ID и метадату.
- **Скупка:** Автоматический прием ресурсов от игроков за валюту сервера.
- **Реальное время в логах:** Использование внешних API для записи точного реального времени операций (timezone: Europe/Kyiv), игнорируя ускоренное время Minecraft.
- **Offline-режим:** Возможность работы без подключения к интернету (данные сохраняются локально).

## 📂 Структура проекта

- `/lua/main.lua` — Ядро и основной цикл магазина.
- `/lua/gui.lua` — Отрисовка графического интерфейса.
- `/lua/me_logic.lua` — Логика взаимодействия с МЭ сетью и сундуками.
- `/lua/network.lua` — Модуль связи с Firebase (REST API).
- `/lua/config.lua` — Конфигурационный файл (Валюта, Админы, Секретные ключи).
- `/lua/installer.lua` — Скрипт автоматической установки в игре.
- `/index.html` — Одностраничное веб-приложение (Админ-панель).
